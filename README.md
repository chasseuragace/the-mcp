## Quick Start

### Basic Usage
```bash
# Run the MCP server with basic configuration
dart run filesystem_mcp_server.dart \
  --read-paths "/Users/username/Projects" \
  --write-paths "/Users/username/Reports" \
  --report-dir "/Users/username/Reports"
```

### MCP Client Configuration
Add to your MCP client configuration:
```json
{
  "mcpServers": {
    "filesystem-intelligence": {
      "command": "dart",
      "args": [
        "run", "/Users/ajaydahal/v4/the_mcp/filesystem_mcp_server.dart",
        "--read-paths", "/Users/ajaydahal",
        "--write-paths", "/Users/ajaydahal/v4/the_mcp/reports"
      ]
    }
  }
}
```
